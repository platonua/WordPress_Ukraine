## Встановлення:

* Встановити модуль можна з маркетплейса Wordpress: Plugins -> Додати новий -> Пошук «Platon Pay» -> Встановити. Або завантаживши архів з цього репозиторія.

* Після успішного встановлення Вам необхідно «Активувати» ваш плагін (Plugins -> Platon Pay -> Активувати).

* В боковому меню адмін консолі вибрати «Platon Pay».

* Ви можете встановити галочку «Тестовий режим» та провести тестування роботи модуля (в тестовому режимі модуль дозволяє повністю перевірити процес оплати на сайті).

* Для подальших налаштувань Вам потрібно зняти «Тестовий режим» і натиснути кнопку «Підключити PSP Platon».

* Заповніть поля «Секретний ключ» та «Пароль».

* Додатково можна вказати «Return url» сторінку на яку платник повернеться після успішної оплати.

* Збережіть зміни.
